# Стикерпаки Telegram в «Случайная картинка»

Можно указывать свои наборы стикеров из Telegram — тогда при нажатии «Случайная картинка» сначала будет выбираться случайный стикер из этих наборов.

## 1. Создать бота и получить токен

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram.
2. Отправьте `/newbot`, придумайте имя и username бота.
3. Скопируйте выданный **токен** (например `123456789:AAH...`).

## 2. Добавить секрет в Supabase

В проекте Supabase:

1. **Project Settings** → **Edge Functions** → **Secrets**
2. Добавьте секрет:
   - **Name:** `TELEGRAM_BOT_TOKEN`
   - **Value:** токен от BotFather

Либо через CLI:

```bash
supabase secrets set TELEGRAM_BOT_TOKEN=123456789:AAH...
```

## 3. Задеплоить Edge Function

```bash
cd birthday-bot
supabase functions deploy get-telegram-sticker
```

## 4. Указать стикерпаки в приложении

1. В генераторе поздравлений найдите блок **«Стикерпаки Telegram»**.
2. Узнайте **имя набора**: откройте стикерпак в Telegram, ссылка вида `t.me/addstickers/ИмяНабора` — нужна часть **ИмяНабора**.
3. Введите имена через запятую, например: `CatHappy, FunnyCats, BirthdayStickers`.
4. Нажмите **«Сохранить»**.

После этого «Случайная картинка» будет сначала брать стикер из ваших наборов. Если стикеры не загрузятся (нет токена или набор не найден), подставится картинка из Reddit или Picsum.
