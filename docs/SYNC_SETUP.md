# Синхронизация контактов между устройствами

Чтобы контакты и сохранённые поздравления отображались и на компьютере, и на телефоне, нужно один раз создать таблицы в Supabase.

## Шаг 1. Выполнить миграцию в Supabase

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard) → ваш проект.
2. В меню слева: **SQL Editor** → **New query**.
3. Скопируйте содержимое файла `supabase/migrations/20250201000000_pozdrav_sync.sql` и вставьте в редактор.
4. Нажмите **Run** (или Ctrl+Enter).

Должны создаться таблицы `pozdrav_contacts` и `pozdrav_congratulations` с политиками RLS (доступ только к своим данным).

## Как это работает

- При **входе** в аккаунт приложение подгружает контакты и сохранённые поздравления с сервера.
- При **добавлении или изменении** контактов/поздравлений данные сохраняются в Supabase и становятся видны на всех устройствах, где вы вошли под тем же аккаунтом.
- Без входа данные хранятся только в этом устройстве (localStorage).
